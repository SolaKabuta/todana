## 🇫🇷 Version Française

### 🌟 Le Projet
Todana est une plateforme de mobilier haut de gamme conçue pour offrir une expérience utilisateur immersive. Ce projet a débuté dans un cadre académique où l'objectif était de simuler un environnement professionnel : travail en équipe, gestion de branches Git, Pull Requests et workflow rigoureux sous React.

Ayant moi-même conçu l'**architecture logicielle** et le **design visuel** de la version originale, j'ai décidé de reprendre le projet en solo pour pousser ma vision plus loin. Mon but : allier mon amour pour l'esthétique à une exigence technique forte, tout en me défiant sur la partie Back-end.

### 🎯 Objectifs
- **Expérience Premium :** Créer une interface fluide et élégante qui valorise le produit.
- **Maîtrise Technique :** Consolider mes acquis sur le Front-end (React/TS) et explorer le développement serveur.
- **Workflow Pro :** Maintenir des standards de code élevés, propres à un environnement de production.

### 🛠 Stack Technique
- **Runtime :** Bun (vitesse et efficacité)
- **Frontend :** React & TypeScript
- **Styling :** Tailwind CSS & Shadcn/UI (Composants modernes et accessibles)
- **Animations :** GSAP (Pour des transitions fluides et haut de gamme)
- **Architecture :** Approche modulaire et scalable.

## 🇺🇸 English Version

### 🌟 The Project
Todana is a high-end furniture platform designed to provide an immersive user experience. This project originated as a collaborative class assignment aimed at replicating a real-world developer workflow: working with multiple branches, handling Pull Requests, and maintaining code integrity in a React environment.

Since I created the **initial design and software architecture**, I chose to develop this personal version to fully realize my creative vision. My goal is to showcase the intersection of my passion for design and coding, while specifically challenging myself with the Back-end implementation.

### 🎯 Goals
- **High-End UX:** Build a fluid, elegant interface that elevates the product presentation.
- **Technical Excellence:** Deepen my Frontend expertise (React/TS) while mastering server-side logic.
- **Professional Workflow:** Adhering to industry standards for clean, maintainable code.

### 🛠 Tech Stack
- **Runtime:** Bun (For high-performance development)
- **Frontend:** React & TypeScript
- **Styling:** Tailwind CSS & Shadcn/UI (Clean, accessible, and modern components)
- **Animations:** GSAP (Crafting "liquid" and premium transitions)
- **Architecture:** Modular and scalable structure.

## 📂 Structure du Projet / Project Structure

- `client/`: React Frontend (Vite + Bun)
- `server/`: Node.js / Express Backend (In Progress)

## 🚀 Installation

```bash
# Clone the project
git clone [repository-url]

# Install dependencies (using Bun)
bun install

# Run the project
bun dev
```
